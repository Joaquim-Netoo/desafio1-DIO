Tags

(voltar)

Em HTML, as tags são elementos que indicam ao navegador como a página deve ser estruturada e exibida.Uma tag normalmente vem em pares: uma de abertura e uma de fechamento.O conteúdo entre as tags de abertura e fechamento é o que será mostrado na página.

Atributos De Tags

(voltar)

Os atributos são informações extras que você adiciona às tags para configurar o comportamento ou aparência de um elemento. Eles são escritos dentro da tag de abertura, e geralmente são compostos por um nome e um valor. Por exemplo, o atributo href define o destino de um link.

  • Listas não ordenadas são usadas quando a ordem dos itens não importa.

Links

(voltar)

Para criar links em HTML, usamos a tag "a". O atributo href especifica o endereço do destino do link, e o conteúdo dentro da tag "a" é o texto clicável.
